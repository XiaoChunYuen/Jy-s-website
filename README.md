# Jy's Channel Portfolio

A personal portfolio website built with Vite, React, TypeScript, Tailwind CSS, Framer Motion, and Supabase CMS.

## Local Development

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Build

```bash
npm run lint
npm run build
```

## Deployment

The recommended deployment target is Cloudflare Pages.

Use:

```text
Build command: npm run build
Build output directory: dist
```

See [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md) for the full deployment checklist.
